# Protocolos Sociales y Señales Conversacionales

Datos de prueba para validar las expresiones regulares de protocolos sociales y señales conversacionales para LATAM y España.

## Estructura

Cada archivo contiene ejemplos de entrada clasificados por categoría y región.

### Formato de archivos:
- `greeting.{region}.txt` - Saludos
- `goodbye.{region}.txt` - Despedidas
- `thanks.{region}.txt` - Agradecimientos
- `affirmation.{region}.txt` - Afirmaciones
- `negation.{region}.txt` - Negaciones
- `uncertainty.{region}.txt` - Incertidumbre

### Regiones:
- `mx` - México
- `ar` - Argentina/Uruguay
- `co` - Colombia/Venezuela
- `es` - España
- `general` - Comunes a todas las regiones
- `typos` - Errores ortográficos comunes
- `extended` - Variaciones con letras repetidas
